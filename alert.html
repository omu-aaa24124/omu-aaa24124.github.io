JavaScript入門：Webページに動きを追加する

概要

Webページを構成する3つの技術

これまでの学習で、Webページを作るための基本的な技術を学んできました。ここで、それらの役割を整理しましょう：

  • HTML： Webページの内容と構造を定義
    • 見出し、段落、リスト、画像などの配置
    • 文書の意味付け
  • CSS（スタイルシート）： Webページの見た目を制御
    • 色、太さ、間隔、位置、配置
    • フォントやレイアウト
  • JavaScript（スクリプト）： Webページに動きや変化を追加
    • ボタンをクリックした時の動作
    • 時間経過による表示の変化
    • ユーザーの入力に応じた処理

JavaScriptでできること

JavaScriptを使うと、静的（変化しない）だったWebページに、以下のような機能を追加できます：

  • 動く時計： 現在時刻を表示し、1秒ごとに更新
  • 画像のスライドショー： 自動的に画像が切り替わる
  • メニューの開閉： クリックするとメニューが表示・非表示
  • フォームの入力チェック： 送信前に入力内容を確認
  • 計算機能： ボタンをクリックして計算結果を表示

今回の目標

この課題では、以下を学習します：

  1. JavaScriptとは何かを理解する
  2. JavaScriptの基本的な書き方を学ぶ
  3. 自分のWebページに「動く時計」を追加する
  4. クライアント側（ブラウザ）でスクリプトが動作することを確認する

JavaScriptとは？

JavaScriptの基本情報

JavaScript（ジャバスクリプト）は、Webページに動的な機能を追加するためのプログラミング言語です。

主な特徴

  • ブラウザ上で動作： サーバーではなく、利用者のブラウザ（クライアント側）で実行される
  • HTMLと組み合わせて使用： HTMLファイルの中に直接書くことができる
  • 学習しやすい： 比較的シンプルな文法で、初心者でも始めやすい
  • 即座に結果が見える： ブラウザで開くとすぐに動作を確認できる

JavaScriptとJavaの違い

注意： JavaScriptとJavaは名前が似ていますが、まったく別の言語です。

項目 JavaScript Java
用途 Webページの動的機能 アプリケーション開発全般
実行環境 ブラウザ Java仮想マシン（JVM）
習得難易度 比較的易しい やや難しい

クライアント側とサーバー側

Webの仕組みを理解するために、クライアントとサーバーの役割を確認しましょう：

  • サーバー側：
    • Webページを保管し、要求に応じて送信する
    • データベースとの連携など
    • 例：PHPやPythonで処理
  • クライアント側（ブラウザ）：
    • 受け取ったHTMLを表示する
    • JavaScriptを実行する
    • 利用者の操作に応じた動作
今回の重要ポイント： JavaScriptはクライアント側（あなたのブラウザ）で動作します。そのため、サーバーへの負荷がなく、即座に反応する動的な機能を実現できます。

JavaScriptの歴史

  • 1995年： Netscape社のブレンダン・アイク氏により開発
  • 当初の名前： LiveScript（後にJavaScriptに改名）
  • 現在： すべての主要ブラウザで標準的にサポート

始めてみよう

JavaScriptを書く場所

JavaScriptをHTMLファイルに含める方法は、主に3つあります：

方法1：HTMLファイル内に直接書く（<script>タグ内）

最も簡単な方法です。HTMLファイルの<head>タグ内または<body>タグ内に書きます。

<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <title>JavaScript練習</title>
    <script>
        // ここにJavaScriptを書く
        alert('こんにちは！');
    </script>
</head>
<body>
    <h1>JavaScriptのテスト</h1>
</body>
</html>

方法2：外部ファイルとして作成（推奨）

JavaScriptを別ファイル（.js）として保存し、HTMLから読み込む方法です。

HTMLファイル（index.html）：

<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <title>JavaScript練習</title>
    <script src="script.js"></script>
</head>
<body>
    <h1>JavaScriptのテスト</h1>
</body>
</html>

JavaScriptファイル（script.js）：

// JavaScriptファイルの内容
alert('こんにちは！');

方法3：HTML要素の属性に直接書く

ボタンなどのイベント（クリックなど）に直接JavaScriptを書く方法です。

<button onclick="alert('クリックされました！')">クリック</button>

最初のJavaScriptプログラム

まずは、ブラウザに簡単なメッセージを表示するプログラムを書いてみましょう。

例1：アラート（警告ダイアログ）を表示

<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <title>初めてのJavaScript</title>
    <script>
        alert('JavaScript が動作しています！');
    </script>
</head>
<body>
    <h1>JavaScriptのテスト</h1>
</body>
</html>

動作： ページを開くと、ダイアログボックスが表示されます。

例2：コンソールにメッセージを表示

<script>
    console.log('コンソールに表示されます');
</script>

確認方法： ブラウザの開発者ツール（F12キー）を開き、「Console」タブでメッセージを確認できます。

例3：HTML要素の内容を変更

<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <title>内容の変更</title>
</head>
<body>
    <h1 id="title">元のタイトル</h1>
    <button onclick="document.getElementById('title').innerHTML = '変更されました！'">
        タイトルを変更
    </button>
</body>
</html>

動作： ボタンをクリックすると、見出しの文字が変わります。

JavaScriptの基本文法

1. コメント（メモ）の書き方

// 1行コメント（この行は実行されません）

/*
  複数行コメント
  このように複数行にわたる
  コメントを書けます
*/

2. 文の終わり

JavaScriptでは、文の終わりにセミコロン（;）を付けます。

alert('こんにちは');
console.log('メッセージ');

3. 大文字と小文字の区別

JavaScriptは大文字と小文字を区別します。

alert('OK');  // 正しい
Alert('NG');  // エラー（Aが大文字）

4. よく使う命令

  • alert('メッセージ'); - ダイアログを表示
  • console.log('メッセージ'); - コンソールに出力
  • document.getElementById('id名') - HTML要素を取得
  • element.innerHTML = '内容'; - 要素の中身を変更

デバッグ（エラー確認）の方法

JavaScriptが動作しない時は、以下の手順で確認しましょう：

  1. 開発者ツールを開く
    • Windowsの場合：F12キーまたはCtrl+Shift+I
    • Macの場合：Cmd+Option+I
  2. Consoleタブを確認
    • エラーメッセージが赤字で表示されます
    • エラーが発生した行番号も表示されます
  3. よくあるエラー
    • スペルミス（alertをAlertと書くなど）
    • セミコロンの付け忘れ
    • 引用符（'または"）の閉じ忘れ

課題：動く時計を作成

課題内容

自分の入口ページ（index.html）の先頭に、現在時刻を表示する「動く時計」を追加してください。

要件

  • 現在の時刻（時：分：秒）を表示すること
  • 1秒ごとに時刻が更新されること
  • 時計が目立つ場所（ページ先頭）に配置されていること
  • 既に作成した内容はそのまま残すこと

実装のヒント

基本的なコード例

<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <title>動く時計</title>
    <script>
        function updateClock() {
            // 現在時刻を取得
            var now = new Date();
            var hours = now.getHours();
            var minutes = now.getMinutes();
            var seconds = now.getSeconds();
            
            // 1桁の場合は0を付ける
            if (hours < 10) hours = '0' + hours;
            if (minutes < 10) minutes = '0' + minutes;
            if (seconds < 10) seconds = '0' + seconds;
            
            // 時刻を表示
            var timeString = hours + ':' + minutes + ':' + seconds;
            document.getElementById('clock').innerHTML = timeString;
        }
        
        // 1秒ごとにupdateClock関数を実行
        setInterval(updateClock, 1000);
        
        // ページ読み込み時に即座に表示
        window.onload = updateClock;
    </script>
</head>
<body>
    <h1>現在時刻</h1>
    <div id="clock" style="font-size: 48px; font-weight: bold; color: blue;">
        00:00:00
    </div>
    
    <!-- ここに既存のコンテンツを配置 -->
</body>
</html>

コードの説明

  • new Date() - 現在の日時を取得
  • getHours() - 時を取得（0～23）
  • getMinutes() - 分を取得（0～59）
  • getSeconds() - 秒を取得（0～59）
  • setInterval(関数, ミリ秒) - 指定した間隔で関数を繰り返し実行
  • 1000ミリ秒 = 1秒

カスタマイズのアイデア

  • 文字の色を変える（style="color: red;"）
  • 文字のサイズを変える（font-size: 64px;）
  • 枠を付ける（border: 2px solid black;）
  • 背景色を付ける（background-color: lightgray;）
  • 日付も表示する（年/月/日）

GitHubでの作業手順

  1. GitHubで自分のリポジトリを開く
  2. index.htmlを編集（鉛筆マークをクリック）
  3. ページ先頭に時計のコードを追加
  4. 「Commit changes」で保存
  5. GitHub Pagesで表示を確認

提出方法

  • 提出物： 動く時計が表示されているページのスクリーンショット（URL欄を含む）
  • 提出先： Moodle上の指定された提出先
  • ファイル形式： Word文書（.docx形式）
  • 提出期限： 1週間後の授業前日 午前中まで
  • 質問方法： Moodleから随時受付

確認事項

提出前に以下を確認してください：

  • 時計が正しく動作している（1秒ごとに更新される）
  • 時刻が正確に表示されている
  • ページの先頭に配置されている
  • 既存のコンテンツが残っている
  • スクリーンショットにURLが写っている
